# Ideaoasis-2
Korean startup idea archive inspired by ideabrowser.com. Curates international business ideas adapted for Korea.

## Features
- Daily idea display from business_ideas.json.
- Archive in business.html.
- Built with HTML/JS for GitHub Pages.

## Setup
- Data: business_ideas.json (from ChatGPT Agent Mode).
- Run: Open index.html in browser or deploy to GitHub Pages.
